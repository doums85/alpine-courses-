x-bind (:) - manipuler les attributs HTML

x-bind(:)
x-bind : Définit la valeur de l'attribut HTML, par exemple value, style, class, etc.

Exemple n°1

Définir la valeur d'un champ de saisie avec un objet value

La valeur d'entrée est liée à "myVar"
Exemple n°2

Définir l'attribut de style

Un peu de texte
La propriété font-weight est définie par "myStyle"
Exemple n°3

Définir l'attribut classes

background-color is set by "myClass"

Étape exemple n°1

  • Dans l'élément Input, définissez x-data="{ myVar: 'nouveau texte'}" puis liez : :value="myVar"

Étape exemple n°2

  • Dans la balise Span, définissez x-data="{ myStyle : 'font-weight:700'}" puis liez : :style="myStyle"

Étape exemple n°3

  • Dans la balise Div, définissez x-data="{ myClass : 'bg-blue-500'}" puis liez : :class="myClass"

Choses à retenir :

  • Utilisez toujours un deux-points (:) à la place de x-bind - c'est plus simple !
